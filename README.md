# Cybersecurity Labs – Purple Team Focus

This repository contains hands-on cybersecurity labs focused on 
attack simulation, detection engineering, and security fundamentals.

## Focus Areas
- Networking & OS fundamentals (aligned with CN and OS coursework)
- Red Team techniques (reconnaissance, web vulnerabilities)
- Blue Team detection (logs, SIEM basics)
- Purple Team workflows (attack → detect → mitigate)
- MITRE ATT&CK mapping

## Lab Methodology
Each lab includes:
- Objective
- Environment
- Attack simulation
- Detection and logs
- Mitigation strategy
- MITRE ATT&CK references

## Tools Used
Wireshark, Nmap, Burp Suite, Splunk, ELK, Linux utilities

## Status
This repository is actively updated as part of my cybersecurity internship preparation.
